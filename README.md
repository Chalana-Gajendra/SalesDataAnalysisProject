# Sales Data Analysis Project

## Overview

This project analyzes an e-commerce sales dataset using Python. The project demonstrates the complete data analysis pipeline, including data loading, cleaning, analysis, visualization, and reporting.

## Features

* Load sales data from a CSV file
* Handle file errors using try-except
* Clean missing and duplicate data
* Calculate sales statistics
* Analyze sales by product and region
* Generate histogram and bar chart visualizations
* Save charts automatically

## Technologies Used

* Python
* Pandas
* Matplotlib
* Visual Studio Code

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Running the Project

Run the following command:

```bash
python main.py
```

The program will:

* Load the dataset
* Clean the data
* Perform analysis
* Display results
* Generate charts
